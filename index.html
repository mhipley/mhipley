Trees Please

In collaboration w/ @linhphaaam, @hackerharker, and @ez-li, this project uses D3 and the Mapbox web services API to manupulate a dataset of trees from DataSF and plot the trees on a map. When a visitor activates their location, they are given walking directions to a random tree, along with the name of the tree and a photo.
